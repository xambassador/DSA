## Count BSTs
Given an integer N, find and return the count of unique Binary search trees (BSTs) are possible with nodes valued from 1 to N.<br>
Output count can be very large, so return the count modulo 10^9+7.
### Input Format :
Integer n 
### Output Format :
Count of BSTs
### Contraints :
1<= N <=1000
### Sample Input 1:
8
### Sample Output 1:
1430
### Sample Input 2:
3
### Sample Output 2:
5
